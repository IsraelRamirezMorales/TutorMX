# Proyecto III - TutorMX (Plataforma de Tutorías)

## Descripción
TutorMX es una aplicación web interactiva que simula una plataforma para encontrar y agendar tutorías académicas. Cumple con los requerimientos de evidenciar programación del lado del cliente mediante la manipulación del DOM, eventos y persistencia de datos..

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica del proyecto.
* **CSS3 & Tailwind CSS:** Diseño responsivo, moderno y estilos externos.
* **JavaScript (ES6):** Lógica del sistema, arreglos, validaciones y manipulación dinámica.
* **LocalStorage:** Persistencia de datos temporales en el navegador.

## Funcionalidades Principales
1. **Autenticación (Simulada):** Formulario de registro con validación de 5 campos y login.
2. **Dashboard Dinámico:** Renderizado de tutores desde un arreglo de objetos JS.
3. **Búsqueda en Tiempo Real:** Filtro de tutores por materia o nombre usando eventos.
4. **Agendamiento:** Modal interactivo para seleccionar fecha, hora y tema.
5. **Gestión de Citas:** Creación y eliminación dinámica de elementos (citas) guardados en el `localStorage`.

## Estructura
El proyecto sigue una estructura modular separando la vista (`.html`), los estilos (`css/`) y los scripts (`js/`).
